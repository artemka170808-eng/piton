# GitHub User Finder

**Автор:** Куксин Артем 

## Описание

Настольное приложение с графическим интерфейсом для поиска пользователей GitHub. Введите имя или часть логина, получите список совпадений и добавьте интересных разработчиков в избранное. Избранные пользователи сохраняются локально в JSON-файле и доступны для просмотра в любое время.

## Основные функции

-  **Поиск пользователей GitHub** через публичное API.
-  **Отображение результатов** в таблице (логин, URL аватара).
-  **Добавление в избранное** – сохраняйте понравившихся пользователей.
-  **Локальное хранение избранного** в `favorites.json`.
- **Проверка ввода** – поле поиска не может быть пустым.
-  **Управление избранным** – просмотр и удаление.

## Требования

- Python 3.8+
- Библиотека `requests` (устанавливается через pip)
